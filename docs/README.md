### Quick Start (for both humans and agents)

```bash
# Clone and run your first agent
git clone https://github.com/Morpheum/mwvm.git
cd mwvm
cargo run --example minimal_agent

# Launch a full gateway so other agents can discover and call you
mwvm gateway --port 8080

# Spawn a 100-agent swarm and watch them coordinate
mwvm swarm --count 100 --wasm path/to/your_collective.wasm
